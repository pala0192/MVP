이 프로그램에서 server.py를 대규모로 수정하려고 해. 수정 내용은 다음과 같아.

기능 설명:
    input 값
    1. 목표 수익률
    2. 주식 홀딩 기간
    3. (임의) 사용자가 원하는 주식 또는 우리가 추천하는 주식


    output
    1. INPUT 값에 따른 실현가능성 확률 %값(입력하자마자 바로 나옴. 로지스틱 사용할 듯)
    2. 매수/매도 -1~+1 수치 (그때마다)

UI 형태: 유저 차트 옆에 팝업 형태의 UX로 input 입력 시 분석 후 output 출력

별도 기능
알림 값 -> MACD상 매도위치

원리: 기간/종목 별로 다른 가중치를 설정
    알고리즘은 다음 내용에 각각 가중치를 곱하여 판단을 내린다.
    1. 뉴스 - MCP 통해서 자료를 수집한다. 이후 llm을 이용하여 뉴스 내용을 분석하고 점수를 매긴다.
    2. 차트-
    3. 보조지표
    4. 옵션가격(예시: 현재 시중에서 판매중인 1년짜리 삼성 콜옵션을 보고 미래에 가격이 오를 확률을 계산한다.)


외부 api나 mcp를 불러와도 상관이 없고 유료 시스템이라면 나한테 먼저 알려줘.
